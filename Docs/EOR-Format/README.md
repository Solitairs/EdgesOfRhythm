## \[音符区]
### 变量
+ **音符索引列表**
  	+ 音符时间 *float*
  	+ 音符索引序号 *int*
  	+ 音符类型 *int*
 + **各类型音符列表**
    + 判定轨道 *int*
    + 初始轨道 *int*
    + 初始速度 *float*
    + 伤害类型 *int*

注:对于轨道（正方体的某个表面的方向）,1~9分别表示上下前后左右  

伤害类型:0无伤 1正常 2加重
